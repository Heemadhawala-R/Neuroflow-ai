# 🚀 NeuroFlow AI

NeuroFlow AI is a modern AI SaaS landing page built as part of the **Frontend Battle Hackathon** using **Next.js**, **TypeScript**, and **CSS Modules**.

The project focuses on creating a responsive, accessible, and visually appealing landing page while following the hackathon requirements.

## ✨ Features

* Responsive landing page
* Hero section with AI dashboard

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/neuroflow-ai.git
```

Move into the project folder

```bash
cd neuroflow-ai
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Create a production build

```bash
npm run build
```
